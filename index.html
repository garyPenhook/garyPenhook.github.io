AVR ASM Navigator

Current package: v0.1.6

AVR ASM Navigator is a local Visual Studio Code extension for Microchip AVR assembly projects. It adds symbol-aware navigation and completion features to `.S` and `.s` files, tuned for MPLAB XC8 AVR workflows and Microchip device-family packs.

What it does

  • Provides hover details for local labels, `.equ` symbols, and DFP-resolved device symbols.
  • Supports go-to-definition, document symbols, workspace symbols, and cross-workspace references.
  • Offers completion for local symbols, AVR instructions, and active-target pack symbols.
  • Includes commands for symbol lookup and active-target inspection inside VS Code.

Why it matters

  • Reduces friction when working in AVR assembly-heavy codebases.
  • Leans on Microchip MPLAB project metadata when available for stronger device auto-detection.
  • Uses installed AVR DFP packs to surface target-specific symbols where developers actually need them.
  • Fits the workflow of developers who want better editor assistance without replacing the Microchip toolchain.

The repository includes VSIX packaging, local checks, and tag-based release automation in GitHub Actions for repeatable distribution.
